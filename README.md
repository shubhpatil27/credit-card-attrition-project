# Credit Card Customer Attrition Prediction System

## Overview

This project builds a **machine learning-based system** to predict customer attrition (churn) in a credit card dataset.

Instead of only performing statistical analysis, we develop a **predictive, explainable, and automated pipeline** that:

* Identifies customers likely to leave
* Explains why they are at risk
* Presents results through a dashboard

This project simulates a real-world solution for a **bank’s customer retention team**.

---

## Business Objective

* Predict which customers are likely to attrite
* Understand key drivers of churn
* Help the business take proactive retention actions

---

## Key Features

* Tiered ML models:

  * Logistic Regression (baseline)
  * Random Forest
  * XGBoost / LightGBM (advanced)
* Proper evaluation:

  * ROC-AUC, F1-score, Recall, PR-AUC
* Model explainability using **SHAP**
* Automated ML pipeline using **Apache Airflow**
* Custom dashboard using **Flask (HTML + CSS)**
* Clean modular project structure

---

## Project Architecture

**Flow:**

Data → Airflow Pipeline → Outputs → Dashboard

* **Airflow (backend)**:

  * Data preprocessing
  * Feature engineering
  * Model training
  * Evaluation
  * SHAP generation
  * Saving outputs

* **Dashboard (Flask)**:

  * Reads outputs from `outputs/`
  * Displays results (no model training here)
